## About

**KRADANKANBAN** is a full-stack Kanban board web application designed to help teams manage tasks and workflows visually and efficiently.

- Developed using **Java Servlets** for backend logic and plain **JavaScript / HTML / CSS** on the frontend
- Supports **task creation**, **priority management**, and **workflow tracking** across board columns
- Structured with a clean **servlet-based architecture** managing data flow between frontend and backend
- UX/UI prototypes designed in **Figma** to ensure intuitive task visualization and usability

---

## Features

### Kanban Board

Visualize your workflow with a board. Tasks are organized into columns representing different stages of progress.

> → [View Frontend source code](https://github.com/Natnarinn/INTEGRATED-US-2-POWERPUFFGIRL-FE)

---

### Task Management

Create, edit, and delete tasks with ease. Each task includes:
- **Title** and **description**
- **Priority level** (Low / Medium / High)
- **Status** tracking across workflow stages

> → [View Backend source code](https://github.com/Natnarinn/INTEGRATED-US-2-POWERPUFFGIRL-BE)

---

### Priority Management

Assign priority levels to tasks so teams can focus on what matters most. Tasks are visually distinguished by priority for quick identification.

> → [View Backend source code](https://github.com/Natnarinn/INTEGRATED-US-2-POWERPUFFGIRL-BE)

---

### Workflow Tracking

Track the lifecycle of every task from creation to completion. Columns represent stages that reflect your team's actual workflow.

> → [View Frontend source code](https://github.com/Natnarinn/INTEGRATED-US-2-POWERPUFFGIRL-FE)

The application's interface was prototyped in **Figma** before development, focusing on:
- Clean and minimal Kanban layout
- Intuitive task creation flow
- Clear visual hierarchy for priority levels
- Responsive column structure for workflow stages
